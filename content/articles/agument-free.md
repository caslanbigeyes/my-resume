### 🌍 地区限制解决方案 - 新增重要说明

> **⚠️ 如果遇到注册地区限制问题**
>
> 如果在注册时出现以下提示：
> `Due to increased demand, we're limiting signups in certain regions to maintain performance for existing customers`
>
> **解决方案**：
> 1. **下载指纹浏览器**：[AdsPower 指纹浏览器](https://www.adspower.net/download)
> 2. **配置代理**：使用梯子（推荐日本、新加坡等地区节点）
> 3. **重要设置**：一定要设置指纹浏览器的 IP 设置的地方走代理
> 4. **验证步骤**：
>    - 访问 Google 搜索
>    - 在 Google 随便搜索内容 如 ip; 在下面的 定位是否正确
>    - 确保google定位不到位置或定位显示为国外 ;我看augment 就使用了 google 定位
>    - 如果显示为国外或无法定位，则设置成功,又能正常的注册了（可以使用自己的域名邮箱，千万不要用什么临时邮箱），我也只能帮到这里了
> 5. **完成后**：在此环境下进行 Augment 账号注册

### ⚡ 快速使用指南

- **换账号**：只需要退出后，正常的官方重新注册登录就行，**不需要清理任何 augment 相关数据**

## 📦 安装方法

### 本地 VSIX 安装包安装

**重要说明**：这是一个本地安装包，需要手动下载并安装，不能通过 VSCode 扩展市场直接安装。

#### 方法一：图形界面安装（推荐，适合一般用户）

1. 访问 [Releases 页面](https://github.com/agassiz/vscode-augment/releases)
2. 下载最新版本 `v0.5091-25-0728` 的 `.vsix` 文件
3. 打开 VSCode，点击左侧扩展图标 (或按 `Ctrl+Shift+X` / `Cmd+Shift+X`)
4. 点击扩展面板右上角的 `...` (更多操作) 按钮
5. 选择 `从 VSIX 安装...` (Install from VSIX...)
6. 浏览并选择下载的 `.vsix` 文件
7. **重要**：安装完成后必须重启 VSCode

#### 方法二：命令面板安装

1. 下载 `.vsix` 文件后，在 VSCode 中按 `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (macOS)
2. 输入 `Extensions: Install from VSIX...`
3. 选择下载的 `.vsix` 文件进行安装
4. **重要**：安装完成后必须重启 VSCode

#### 方法三：命令行安装（适合开发者）

```bash
# 下载 .vsix 文件后，在文件所在目录执行：
code --install-extension vscode-augment-0.5091.vsix
```

#### 方法四：直接拖拽安装（最简单）

1. 下载 `.vsix` 文件到本地
2. 打开 VSCode 扩展面板 (`Ctrl+Shift+X` / `Cmd+Shift+X`)
3. 将 `.vsix` 文件直接拖拽到扩展面板中
4. 确认安装并重启 VSCode

> **💡 安装提示**：推荐使用方法一（图形界面）或方法四（拖拽），这两种方式对一般用户最友好，不需要使用命令行。

## 🚀 使用说明

**本地安装包特点**：
- 这是一个修改版的本地安装包，不依赖官方扩展市场
- 安装完成后**必须重启 VSCode** 才能正常使用
- 功能与官方版本完全一致，只是移除了部分风控代码

安装完成后，插件将自动激活。您可以通过以下方式使用：

1. **功能访问**：通过命令面板 (`Ctrl+Shift+P` / `Cmd+Shift+P`) 搜索相关命令
2. **快捷键**：查看插件设置中的快捷键配置
3. **设置配置**：在 VSCode 设置中搜索 "augment" 进行个性化配置

插件提供了丰富的配置选项，您可以在 VSCode 设置中进行调整：

- 搜索 "augment" 查看所有可配置项

## 💡 重要使用说明

### 📧 邮箱要求
- **必须使用正常邮箱**：Gmail、Outlook、QQ邮箱等主流邮箱服务
- **禁止使用临时邮箱**：如 2925.com、10minutemail 等临时邮箱服务
- **风控提醒**：临时邮箱可能触发系统风控机制，导致功能异常
