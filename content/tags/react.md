# React

React 是由 Facebook 开发的用于构建用户界面的 JavaScript 库。它采用组件化的开发模式，让开发者能够构建可复用、可维护的 UI 组件。

## 核心特性

- **组件化**: 将 UI 拆分为独立、可复用的组件
- **虚拟 DOM**: 提供高效的 DOM 更新机制
- **单向数据流**: 数据从父组件流向子组件
- **JSX**: JavaScript 的语法扩展，让编写组件更直观
