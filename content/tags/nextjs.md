# Next.js

Next.js 是一个基于 React 的全栈框架，提供了生产级别的功能，如服务端渲染、静态站点生成、API 路由等。

## 主要特性

- **App Router**: 新一代路由系统
- **服务端渲染 (SSR)**: 提升 SEO 和首屏加载速度
- **静态站点生成 (SSG)**: 构建时预渲染页面
- **API 路由**: 内置 API 开发能力
- **图片优化**: 自动图片优化和懒加载

## 适用场景

- 企业级 Web 应用
- 电商网站
- 博客和内容网站
- 营销页面
